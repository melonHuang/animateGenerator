# 欢迎使用Animation Generator

Animation Generator是一个集成在Chrome开发者工具中的扩展程序。她旨在帮助web开发者轻松迅速地为页面上的元素添加动画。通过她，开发者可以为页面元素选择动画效果，动画时长，Timing function等，并实时预览。最终，Animation Generator会根据你的选项，生成兼容多浏览器的CSS动画代码。
